# 🧮 3D Calculator

A modern *3D Calculator* built using HTML, CSS, and JavaScript with a clean UI and interactive buttons.

🔗 *Live Demo:* https://3-d-calculator-khaki.vercel.app  
🔗 *GitHub Repo:* https://github.com/snehasharm2004-dotcom/3D-Calculator

---

## ✨ Features

✅ Beautiful 3D button design  
✅ Responsive layout (mobile + desktop)  
✅ Basic arithmetic operations  
- Addition (+)  
- Subtraction (−)  
- Multiplication (×)  
- Division (÷)  
✅ Clear (AC) & Delete (DEL) functions  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3 (3D effects & styling)  
- JavaScript (logic & functionality)
